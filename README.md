# EduDash

Educational Management System - A Vue 3 dashboard for managing educational units, students, tasks, and labels.

## Requirements

- Node.js 18+ 
- npm or yarn

## Installation

```bash
npm install
```

## Running

```bash
npm run dev
```

The application will be available at `http://localhost:3000`
